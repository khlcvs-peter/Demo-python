簡短說明
- 這個專案包含一個用於下載 Google Drive 資料夾並讀取 docx 的範例腳本 (catch-doc-net.py)。

上傳到 GitHub 的步驟（快速）
1. 在本地檔案夾執行 upload_to_github.py，或手動執行：
   git init
   git add .
   git commit -m "Initial commit"
   git remote add origin <你的遠端 repo URL>
   git push -u origin main

備註
- upload_to_github.py 會要求你輸入遠端 repo URL（例如 https://github.com/username/repo.git）。
- push 需要你已經在系統上設定好 Git 的認證（token 或 SSH key）。
